# Homepage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the complete JobPilot landing page as a static, server-rendered page matching `context/designs/landing-page.png`, with all CTAs linking to `/login`.

**Architecture:** Six pure Server Components assembled in `app/page.tsx`. No client JavaScript. `Navbar` and `Footer` are reusable layout components; `Hero`, `Features`, `Testimonial`, and `BottomCta` are homepage-specific. All accent colors and surfaces come from project tokens — zero hardcoded hex.

**Tech Stack:** Next.js 16 App Router, Tailwind CSS v4 (tokens via `@theme` in `app/globals.css`), `next/image`, `next/link`, `lucide-react` (icons), Inter font (already wired in root layout)
